Project 7: Comprehensive data analytics

While preparing for the 2020 DSN competition, I decided to play around with the 2019 DSN competition dataset. I performed full EDA, which includes univariate analysis, bivariate analysis and multivariate analysis. I also cleaned the dataset with sophisticated data cleaninig procedures. The most amazing thing wa that, during my analysis, I discovered inbalance in the target variable. Although, I didnt apply any balancing technique to balance the dataset, I stated it at the end of the project that further research was going to involve using SMOTE to balance the dataset.

During my NYSC, I started a tutorial class for students preparing for WAEC(West Afican Certificate Examinations), it made me fall in love with maths so much that I made up my mind that I was going to use advanced maths to solve more challenging problems. Project 4: Prediction of the prices of used phones.

As we know, in a country like Nigeria, some of the phones we buy are used phones. We can purchase phones like these even from online stores like jiji. Anyone familiar with jiji plattform knows that the sellers give brief description about the phones they want to sell and the location they are selling from. This model used these features to give predictions of used phones based on those descriptions given by the seller.
